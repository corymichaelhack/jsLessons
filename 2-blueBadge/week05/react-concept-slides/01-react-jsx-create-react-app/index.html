JSX & Create-React-App

Getting Up and Running with React

JSX

What is JSX?

JSX is a syntactic extension for JavaScript. It is used by many React developers for their markup code. Markup code is simply the code that generates what we see on a website. HTML is markup code we've already been using.

JSX vs. HTML


            const title = <h2>Hello React!</h2>;

JSX looks very close to HTML, but it is really still JavaScript. Under the hood, React is reading our JSX, compiling it into a function call, and building out HTML elements for us. So we're free to write JSX without having to worry about how it'll be read by the browser.

Embedding Data


            const appTitle = 'EFA Chat';
            const title = <h2>{ appTitle }</h2>;

React and JSX also give us an easy way to embed regular JavaScript expressions. React will attempt to parse anything between two curly braces as a JavaScript expression, and output the result in the place of the curly braces.

JSX as an Expression


            if(loggedIn) {
                return <h2>Welcome back { username }</h2>;
            } else {
                return <h2>Please log in</h2>;
            };

Since JSX is an expression, you can use it inside of if statements, and for loops, or even return it from functions.

Attributes with JSX

Like HTML elements, tags used in JSX can accept attributes, but there are a few things to keep in mind.

Attribute Values


            const avatar = <img src={ imgPath }></img>;
            const img = <img src="https://www.google.com"></img>;

Instead of wrapping our data values for attributes with quotes as we'd do in HTML, we wrap expression values with curly braces, but non-data values can still use quotes.

Attribute Names


            <button onClick={this.handleSubmit}>Save</button>

We must also be aware that instead of leaving attribute names all undercase, when writing JSX we camel case them.

Create React App

What is C.R.A?

Create-react-app is the name of a very handy package created by Facebook. It's intent is to get you up and running with React as quickly as possible.

Why use C.R.A?

Create React App is what is known as a build workflow tool. It creates and initializes a basic React application, configures out settings, and sets up a development server with live reload. Not all of these are standard React features. So we're very lucky Facebook made and released this tool.
