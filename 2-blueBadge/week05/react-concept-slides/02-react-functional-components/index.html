Functional Components

What is a functional component?

A functional component, also known as a display component. Is simple a function, that returns some JSX for React to render into our Apps. Functional components are capable for logic, however they can't use state, so they usually leave the logic to stateful components.

Creating Functional Components


    const FunctionalComponent = (props) => {
        return (
                <div>
                </div>
                )
            }

The basic skeleton for a functional component is just an arrow function. Since it's just a function, it can take props as an argument.

Using Props


    const FunctionalComponent = (props) => {
        return (
                <div>
                    <h2>{ props.title }</h2>
                </div>
                )
            }

Since we can pass in props as an argument, this functional component gets access to the props object, and we are free to use those values here.

Using Logic


    const FunctionalComponent = (props) => {
        const title = props.title ? props.title : 'Hello React';
        return (
                <div>
                    <h2>{ title }</h2>
                </div>
                )
            }

Functional Components can still contain logic, however you usually won't see any functions being defined. We can define values above the return, and use them in the JSX we return.
